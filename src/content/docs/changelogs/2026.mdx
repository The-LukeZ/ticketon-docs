## 20.04.2026

### Starter Messages Respect Custom Message Allowed Mentions

If your ticket starter message uses a custom message, allowed mentions are now pulled from that custom message config.
By default, all pings except for the support roles and the ticket author are disabled. So if you're mentioning a role in your starter message and wondering why it doesn't ping — that's why probably.
Configure it in the allowed mentions of your custom message and it'll work.

### Premium Page is Back

The premium page was broken. It's fixed now.

## 17.04.2026

### New

- New dashboard! Now it's easier to use, looks better and has more feature! [Check it out!](https://ticketon.app/dash)
- Allowed Mentions of Custom Messages are now used when the message is a starter message. [Read more](/dashboard/custom-messages#allowed-mentions).
- Automation is also implemented
- Dashboard Access control! You can now control which members of your server can access the dashboard and manage the bot's settings. [Read more](/guides/access-control).

### Changes

- Preserved Images are no longer "in development", they are now removed from the docs and legal documents, because the legal implications of preserved images are too much for the feature to be worth it.
  If you have any questions or ideas regarding this, please join the support server and let me know!
- The Message Builder got an upgrade!

### Fixes

A lot.

## 24.02.2026

Thread creation message now features two buttons: One to join the ticket thread and one to just view the ticket channel without joining it.

_A thread creation message is sent when you have **Use Threads** enabled in a ticket category._

## 31.01.2026

### Debug Command!

You can now use `/debug` to check if the bot has all required permissions for a ticket category.

## 22.01.2026

### More message importing!

You can now import messages directly from Discord in the dashboard! Just head to the **Custom Messages** page,
click the **Import** Button and change the tab in the dialog to **Discord Message**.

<ImageWrapper>
  ![Import Messages from Discord
  Example](../../../assets/docs/changelogs/2026/import-from-discord.png)
</ImageWrapper>

This feature should allow quicker setting up of your server with existing messages from a previous bot or configuration,
however do note that embeds are not supported as the structure is fundamentally different between [Components V2](/guides/components-v2) and (old) Embeds.

## 20.01.2026

- New `/config set-language` command to set the bot's language for your server. [Read more](/commands/overview#set-language).
- Many minor bug fixes and one major update (behind the scenes).
